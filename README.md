## Опис проєкту
Цей репозиторій створено для демонстрації навичок роботи з системою контролю версій Git та налаштування базового CI/CD за допомогою GitHub Actions.

## Мета роботи
Навчитися працювати з розгалуженнями (branching), виконувати коміти, створювати Pull Requests та автоматизувати перевірку коду.

## Інструкція запуску
Щоб запустити скрипт локально:
1. Відкрийте термінал у папці проєкту.
2. Виконайте команду: `python src/app.py`
